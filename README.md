# Multiple-disturbance
This project intended to analyze the impacts from multiple forest disturbance using tree ring.

There are five python 2.7 programs in total. We just need to run them following the order. The size of the data source were big (3G). The data source and instructions for the codes can be downloaded via the link: https://drive.google.com/drive/folders/1CSlxEPark7QQHC0AE6e76wYlM-6e3X-6?usp=share_link
